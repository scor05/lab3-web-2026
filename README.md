# HTML Pick-Your-Own-Adventure

Este laboratorio es una historia corta escrita totalmente en HTML que tiene el
formato clásico de "pick-your-own-adventure". La historia se puede ver en [joelsiervas.online/24472/lab3/](joelsiervas.online/24472/lab3)

En este repositorio se encuentra un `codegenerator.py`, el cual no se utilizó para
nada en la página final de por sí, sino que fue solo utilizado para generar varios
archivos `index.html` que se repiten dentro de la carpeta de `codes/`. El propósito 
de ellos se explica más a detalle en la página principal de la historia (para no 
dar spoilers desde aquí :D).
